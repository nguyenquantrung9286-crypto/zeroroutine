RULE 2: All buttons must work
→ Каждая кнопка и ссылка делает что-то конкретное:
   - CTA-кнопки → modal
   - Nav-ссылки → smooth scroll к секции
   - Telegram → href к реальному URL
   - "Подробнее" → modal с описанием продукта

RULE 3: Russian language only
→ Весь контент страницы на русском. Код — на английском.

RULE 4: No placeholders
→ Никакого Lorem ipsum. Только реальный контент по бренд-гайду.

RULE 5: Mobile-first responsive
→ Сайт должен корректно выглядеть на 375px, 768px и 1440px.


RULE 8: Accessibility
→ Все интерактивные элементы доступны с клавиатуры.

→ Сайт работает только в светлой теме (не нужен переключатель).

RULE 10: SEO basics
